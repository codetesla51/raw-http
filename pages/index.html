Welcome to Uthman's Web Server

This is a custom HTTP server built from scratch using Go. No frameworks used - just pure TCP socket programming and HTTP protocol implementation.

Source Code Repository Uthman's Website

Available Routes:

  • Current Server Time
  • Hello Page
  • Static File Example

Server Features:

  • Concurrent connection handling with goroutines
  • HTTP request parsing and header processing
  • File-based static content serving
  • Dynamic route handling
  • User-Agent detection
